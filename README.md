# Dynamic Marks

动态收藏夹管理工具 ——整理收藏夹只需一键。

## 功能

- **按热度排序** — 根据浏览历史统计访问频次，一键将常访问的书签排到前面
- **发现未收藏** — 找出高频访问但未收藏的网站，提醒添加
- **清理无用书签** — 标记从未访问或长期未使用的书签，帮助清理

## 安装

1. 打开 `chrome://extensions`
2. 开启右上角 **开发者模式**
3. 点击 **加载已解压的扩展程序**
4. 选择本项目文件夹

## 使用

点击浏览器工具栏的扩展图标，在弹出窗口中切换标签页：

| 标签 | 操作 |
|------|------|
| 排序 | 点击按钮，自动按访问频次重排所有书签 |
| 发现 | 选择时间范围，查看高频但未收藏的网站，点「+ 收藏」添加 |
| 清理 | 选择筛选条件，查看无用书签，点「移除」删除 |

## 权限

- `bookmarks` — 读取和修改书签
- `history` — 读取浏览历史

## 项目结构

```
├── manifest.json      # 扩展配置

```
